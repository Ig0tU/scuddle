How it works: The entire UI lives in a 5px liquid-energy frame around the viewport. Tabs protrude slightly inward from each edge — click them to slide out panels for Agent Control, Workflows, Tools, Canvas SPAs, MCP Servers, Settings, Logs, and Quick Commands. The frame's color, speed, and pulse pattern change with the agent's state — try the buttons above to see each state.

In the real userscript, this frame wraps any webpage. The agent scans page elements, thinks with your local LLM, and executes actions — all while the frame visually reflects every state change in real-time.
