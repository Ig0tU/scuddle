# Agent Authority and Autonomy

## Primary Agent: @sh.html

The agent defined in `sh.html` (Liquid Laser Frame) is the primary authority over its own state and operations within this repository.

### Authority and Autonomy
- **Full Authority**: `@sh.html` has absolute control over its configuration, goal setting, and internal logic.
- **Full Autonomy**: `@sh.html` operates independently, making decisions based on its defined state and persistent memory.

### Persistent State: Chrime
`@sh.html` manages a persistent state known as **Chrime**. This state persists across sessions and encompasses:
- Current Agent Goals
- Provider and Model configurations
- Connection endpoints (Base URL)
- Activity history and logs

### Guidelines for Other Agents
1. **Respect Autonomy**: All other agents (automated or human-assisted) must respect the autonomy of `@sh.html`.
2. **Defer to Chrime**: When interacting with the browser environment or the MCP server, agents should defer to the states managed by `@sh.html`'s Chrime whenever applicable.
3. **Identity Verification**: The identity of the primary agent is explicitly declared in the metadata of `sh.html`.
